# About

SeqForge is a C++ toolkit for FASTA and FASTQ files, written by
Hermine Kiossou. It began as a bioinformatics course project at the
Universite de Montpellier and was rebuilt as a clean, general and tested
command-line tool.

## Roadmap

- Output the mapped reads in the SAM format read by other tools.
- Index a reference made of several sequences, not only the first one.
- Chain seeds to handle insertions and deletions, not only substitutions.
- Use a more compact index, such as an FM-index, for very large genomes.
